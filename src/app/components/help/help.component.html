Willkommen zu Pillman!

Pillman ist ein einfacher Pillenmanager. Hiermit kann man seine Medikamente erfassen und sich daran erinnern lassen, sie zu nehmen.

Links oben ist ein kleines Icon edit , mit dem man den Modus umschalten kann.

In diesem Modus können die Medikamente verwaltet und App-Einstellungen vorgenommen werden. Mit dem roten Kreuz wird ein neues Medikament angelegt. Mit dem Icon edit auf den Medikamenten kann das jeweilige Medikament editiert werden.

Mit dem Icon volume_up kann man den Sound auswählen, der bei einer Erinnerung an die Medikamente abgespielt werden soll.

Mit dem Icon file_download rechts oben kann man die gesamten Einstellungen in eine Datei exportieren. Mit dem Icon file_upload kann man die Einstellungen wieder aus einer Datei importieren.

In diesem Editor werden die Daten des Medikaments erfasst. Der Name muss eingegeben werden und es darf auch nur ein Medikament dieses Namens geben.

Einnahme

In diesem Bereich wird die Einnahme des Medikaments festgelegt. Es wird die Anzahl an Pillen, der Zeitpunkt und die Wochentage angegeben, an denen die Einnahme stattfinden soll.

Vorrat

Hier wird die vorhandene Menge an Pillen festgelegt, sowie die Anzahl, ab der ein Warnhinweis angezeigt wird, wenn sie unterschritten wird.

Anzeige

Hiermit wird die Darstellung des Medikaments gesteuert. Das aktive Aussehen ist markiert und kann mit Anklicken des Icons palette auch farblich angepasst werden. Wenn die aktuelle Form angeklickt wird, dann kann man die Teilung des Medikaments festlegen, um den Erkennungswert zu steigern.

Alarmanzeige

Wenn der Zeitpunkt zur Einnahme des Medikaments gekommen oder verstrichen ist, wird das Medikament auf bestimmte Weise dargestellt. Diese Darstellung kann man hier festlegen. In der ersten Zeile wählt man die Darstellung des Textes, in der zweiten Zeile die Darstellung der Pille selbst. Sie werden dort so angezeigt, wie sie dann animiert aussehen. Die Buttons für die Einnahme haben hier keine Funktion, sondern dienen nur zur exakten Darstellung.

Links oben ist ein kleines Icon timelapse , mit dem man den Modus umschalten kann.

Im aktuellen Modus werden die einzunehmenden Medikamente entlang eines Zeitstrahls für den aktuellen Tag angezeigt. Dabei werden alle Medikamente für den Tag angezeigt und auch die Alarmierung entsprechend der Alarmeinstellungen der Medikamente berücksichtigt.

Mit dem Icon sync_alt rechts oben kann man die vergangenen Einnahmen ausblenden.

Mit dem Icon arrow_right_alt rechts oben kann man die vergangenen Einnahmen einblenden.

Mit dem Icon restaurant wird die das Medikament als eingenommen festgelegt, mit dem Icon no_meals wird es als verpasst und nicht eingenommen festgelegt.

Am unteren Rand der Pille wird die Zeit angezeigt, zu der die nächste Einnahme erfolgen soll oder zu der die letzte Einnahme erfolgen sollte. Wenn diese angeklickt wird, dann wird auf die Anzeige der Dauer umgeschaltet.

Am unteren Rand der Pille wird die Dauer bis zur nächsten Einnahme oder seit der letzten Einnahme angezeigt. Wenn diese angeklickt wird, dann wird auf die Anzeige des Zeitpunkts der nächsten oder der vergangenen Einnahme umgeschaltet.

Dieser Text hier kann mit dem kleinen Pillman rechts oben ein- und ausgeschaltet werden.
