Willkommen zu Pillman!

Pillman ist ein einfacher Pillenmanager. Hiermit kann man seine Medikamente erfassen und sich daran erinnern lassen, sie zu nehmen.

Links oben ist ein kleines Icon edit , mit dem man den Modus umschalten kann.

Im aktuellen Modus können die Medikamente verwaltet und Einstellungen vorgenommen werden. Mit dem roten Kreuz wird ein neues Medikament angelegt. Mit dem Icon edit auf den Medikamenten kann das jeweilige Medikament editiert werden.

Mit dem Icon volume_up kann man den Sound auswählen, der bei einer Erinnerung an die Medikamente abgespielt werden soll.

Mit dem Icon file_download rechts oben kann man die gesamten Einstellungen in eine Datei exportieren. Mit dem Icon file_upload kann man die Einstellungen wieder aus einer Datei importieren.

In diesem Editor werden die Daten des Medikaments erfasst. Der Name muss eingegeben werden und es darf auch nur ein Medikament dieses Namens geben.

Einnahme

In diesem Bereich wird die Einnahme des Medikaments festgelegt. Es wird die Anzahl an Pillen, der Zeitpunkt und die Wochentage angegeben, an denen die Einnahme stattfinden soll.

Es können mehrere Einnahmen eines Medikaments erfasst werden. Dazu das Icon add unterhalb der letzten Einnahme anklicken und es erscheint eine neue Zeile, in der der neue Einnahmezeitpunkt erfasst werden kann.

Mit dem Icon delete in der Zeile der Einnahme kann diese gelöscht werden. Die oberste Einnahmezeile kann nicht gelöscht werden.

Vorrat

Hier wird die vorhandene Menge an Pillen festgelegt, sowie die Anzahl, ab der ein Warnhinweis angezeigt wird, wenn sie unterschritten wird.

Anzeige

Hiermit wird die Darstellung des Medikaments gesteuert. Das aktive Aussehen ist markiert und kann mit Anklicken des Icons palette auch farblich angepasst werden. Wenn die aktuelle Form angeklickt wird, dann kann man die Teilung des Medikaments festlegen, um den Erkennungswert zu steigern.

Alarmanzeige

Wenn der Zeitpunkt zur Einnahme des Medikaments gekommen oder verstrichen ist, wird das Medikament auf bestimmte Weise dargestellt. Diese Darstellung kann man hier festlegen. In der ersten Zeile wählt man die Darstellung des Textes, in der zweiten Zeile die Darstellung der Pille selbst. Sie werden dort so angezeigt, wie sie dann animiert aussehen. Die Buttons für die Einnahme haben hier keine Funktion, sondern dienen nur zur exakten Darstellung.

Im aktuellen Modus werden die einzunehmenden Medikamente entlang eines Zeitstrahls für den aktuellen Tag angezeigt. Dabei werden alle Medikamente für den Tag angezeigt und auch die Alarmierung entsprechend der Alarmeinstellungen der Medikamente berücksichtigt.

Links oben ist ein kleines Icon timelapse , mit dem man den Modus umschalten kann.

Mit dem Icon sync_alt rechts oben kann man die vergangenen Einnahmen ausblenden.

Mit dem Icon arrow_right_alt rechts oben kann man die vergangenen Einnahmen einblenden.

Mit dem Icon restaurant wird das Medikament als eingenommen festgelegt, mit dem Icon no_meals wird es als verpasst und nicht eingenommen festgelegt. Wenn es sich um eine Salbe handelt, werden die Icons clean_hands und do_not_touch verwendet.

Am unteren Rand der Pille wird die Zeit angezeigt, zu der die nächste Einnahme erfolgen soll oder zu der die letzte Einnahme erfolgen sollte. Wenn diese angeklickt wird, dann wird auf die Anzeige der Dauer umgeschaltet.

Am unteren Rand der Pille wird die Dauer bis zur nächsten Einnahme oder seit der letzten Einnahme angezeigt. Wenn diese angeklickt wird, dann wird auf die Anzeige des Zeitpunkts der nächsten oder der vergangenen Einnahme umgeschaltet.

Das Alarmmenü zeigt die verfügbaren Sounds an. Von diesen Sounds kann einer aktiviert werden, um an die Einnahme von Medikamenten zu erinnern. Mit dem Icon play_arrow wird der entsprechende Sound abgespielt, um festzustellen, ob es der Sound der Wahl ist.

Während der Sound abgespielt wird (einige sind etwas länger), verändert sich das Icon in stop . Wenn dieses angeklickt wird, dann wird der Sound beendet.

Mit dem Anklicken des Namens wird der Sound als Erinnerungston festgelegt,.

Hier kann eine Farbe gewählt werden. Es gibt verschiedene Modi, die man mit dem kleinen Icon links oben umschalten kann. Am unteren Rand wird links die aktuelle Farbe angezeigt, die mit dem Dialog geändert werden soll. Daneben befinden sich ein bis zehn Farbfelder, die durch Auswahl einer Farbe erweitert werden. Durch Anklicken eines der Farbfelder wird die Farbe dann dort gesetzt, wo der Dialog gestartet wurde.

Der aktuelle Modus ist:

blender Farbmischer

In diesem Modus kann man sich die Farbe zusammen mischen. Die runden Ecken des angezeigten Feldes werden mit einer Farbe versehen und das Feld zwischen diesen Ecken wird entsprechend der Farbauswahl mit einem Farbverlauf befüllt. Wenn eine Ecke angeklickt wird, dann erscheint ein Farbrad, aus dem man die Farbe für diese Ecke auswählen kann.

Die Farben sind vorgegeben und repräsentieren die hellsten Farben, schwarz und weiss. Wenn man das Farbfeld anklickt, wird zusätzlich dazu, dass die Farbe in die untere Farbleiste übernommen wird, in der Mitte oben im Farbrad diese Farbe eingefügt. Dadurch kann man diese Farbe auch als Farbe für eine Ecke auswählen und so verschiedene Mischungen vornehmen. Die Farbe wird beim nächsten Anklicken einer Farbe ersetzt, so dass immer maximal neun Farben zur Auswahl stehen.

Mit dem Mausrad kann das Farbfeld grobkörniger (Mausrad runter) oder feiner (Mausrad hoch) eingestellt werden. Damit kann man eine genauere Farbauswahl treffen.

image Bild

In diesem Modus wird die Farbe aus einem Bild ermittelt. Das Bild kann mit dem Icon upload rechts unten geladen werden. Wenn man dann den Cursor über das Bild bewegt, wird eine Lupe angezeigt, die den Bereich unter dem Cursor heranzoomt. Dieser Zoomfaktor kann durch das Mausrad verändert werden. Damit ist es möglich die Farbe, die man gerne möchte besser zu lokalisieren. Wenn der Zoomfaktor ausgeschaltet wurde (Mausrad nach oben), dann verschwindet die Lupe, kann aber mit Mausrad runter wieder angezeigt werden.

Das Bild wird im Browser gespeichert und steht beim nächsten Aufruf der Seite wieder zur Verfügung. Es muss also nicht jedesmal neu geladen werden.

palette RGB

In diesem Modus wird die Farbe mit Reglern für RGB (Rot, Grün, Blau) und HSL (Hue, Saturation, Lightness) eingestellt.

Die aktuelle Farbe, die sich unter dem Mauscursor befindet wird immer in der Farbleiste am unteren Rand im letzten Farbfeld angezeigt.
